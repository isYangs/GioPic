# @giopic/core

> GioPic 插件系统核心模块

## 概述

`@giopic/core` 是 [GioPic](https://github.com/isYangs/GioPic) 图床管理工具的插件系统核心模块，为所有插件提供必需的通用功能。

**本模块由 GioPic 主程序自动加载，无需手动安装。**

## 特性

- **HTTP 请求封装** - 统一的请求处理和错误管理
- **日志系统** - 结构化日志记录，便于调试和监控
- **插件数据存储** - 插件配置和数据的持久化方案
- **类型定义** - 完整的 TypeScript 类型支持

## 插件开发

如需开发 GioPic 插件，请参考 [GioPic 插件开发文档](https://github.com/isYangs/GioPic)。
